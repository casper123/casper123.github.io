A small introduction about my self

Abdul Wahab Kotwal - Casper

Web & Mobile Application Developer | Founder at SandBox - Coworking Space for freelancers and startups

Hi, I'm Abdul a.k.a casper, I've been coding since I was 12. I aim to develop most interesting mobile and web apps for my clients.

I've been developing web and mobile apps for last 5 years. I enjoy working with MEAN Stack, I'm in love with front-end development in AngularJS. I also work on hybrid application frameworks like Ionic, Framework7 and OnsenUI.

I use Git for code hosting. Gulp is my favorite when it comes to building apps.

I work on back-end using LAMP Stack. I'm profficiant in procedural PHP and its frameworks like yii, laravel and codeigniter. I use mySQL for databases and Sphinx search API where full-text search is required.

I do DevOps too. I can setup linux and nodejs servers and keep them running.

Other then that, I have great command on jQuery, KineticJS, React, EmberJS, HTML5, CSS3 (scss), MongoDB, Firebase APIs and Wordpress.

p.s: I am not a designer or a manager.Just the programming for me, thanks.
